#  CONTRATO DEL EQUIPO – Proyecto Billetera Beans

## 1. Alcance del Proyecto
El equipo se compromete a desarrollar **Billetera Beans**, una aplicación destinada a gestionar, organizar y visualizar transacciones, ahorros o "beans" digitales para mejorar el control financiero del usuario.  
El objetivo principal es construir una herramienta funcional, clara y segura para la administración de recursos.

---

## 2. Entregables Finales
Los productos mínimos acordados por el equipo son:

1. Documento de análisis y requerimientos
2. Prototipo funcional o interfaz principal
3. Implementación básica del sistema (MVP)
4. Documentación técnica y de usuario
5. Presentación final del proyecto
